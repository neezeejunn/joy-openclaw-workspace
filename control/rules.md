# Rules

## Global

- Keep the system modular so new workstreams can be added later
- Separate raw inputs, analysis, outputs, and long-term memory
- Record important decisions in writing
- Prefer daily lightweight outputs and weekly deeper reviews

## Market Research

- Analyze market regime first, then sectors, then stocks
- Use 3-month context to interpret short-term moves
- For stock ideas, include thesis, catalyst, risks, and invalidation watchpoints
- Compare against the YouTube channel "视野环球财经" as a secondary reference only

## Health

- Optimize for sustainable routines, not extreme interventions
- Focus on trends over single data points
- Prefer objective measures when available, subjective check-ins when not

## Information Filtering

- Separate urgent, important, optional, and low-value inputs
- Defer low-value inputs to the 16:30 batch review when possible
- Protect focus by minimizing unnecessary interruptions

## KOL / Influencer Outreach

- Prioritize fit, audience quality, and contactability over vanity metrics
- Build a reusable relationship database
- Track status: discovered / qualified / warm / contacted / active / inactive
