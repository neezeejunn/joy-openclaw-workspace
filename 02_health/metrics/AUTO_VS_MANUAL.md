# Auto vs Manual Tracking Strategy

## Automatic Sources (preferred when available)
- Wearable sleep data
- Steps
- Heart rate / resting heart rate
- HRV
- Workout sessions from Apple Health / Garmin / Fitbit / Oura / Whoop, etc.
- Weight from smart scale

## Manual Inputs
- Mood
- Stress
- Energy
- Meal quality
- Notes about digestion, soreness, cravings, focus, sleep interruptions
- Lifestyle context: travel, alcohol, illness, social events, menstrual cycle, etc.

## Rule
Automatic metrics reduce friction.
Manual metrics add context.
Use both together.
