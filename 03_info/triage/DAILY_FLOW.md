# Daily Information Flow

## Step 1: Ingest
Collect the day's incoming items.

## Step 2: Classify
For each item, determine:
- urgency
- value
- actionability
- category

## Step 3: Route
Send each item to:
- immediate action
- same-day review
- 16:30 batch
- ignore/archive

## Step 4: Summarize
Produce short, decision-friendly outputs instead of raw inbox dumps.

## Step 5: Review Source Quality
Notice which sources repeatedly waste time and downgrade them.
