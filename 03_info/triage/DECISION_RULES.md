# Information Triage Decision Rules

## Routing Categories
### 1. Immediate
Use for:
- urgent
- high-value
- decision-relevant
- deadline-sensitive

### 2. Same-Day Review
Use for:
- useful but not urgent
- may affect today's planning

### 3. 16:30 Batch
Use for:
- low urgency
- low or medium value
- non-disruptive reading

### 4. Ignore / Archive
Use for:
- repetitive noise
- promotions with no real value
- low-relevance updates

## Core Rule
Interrupt attention only when the likely value exceeds the cost of interruption.
