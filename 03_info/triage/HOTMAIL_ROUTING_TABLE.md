# Hotmail Routing Table

## Immediate
- Shipping / delivery / tracking / pickup / delay
- Work cooperation / business / contracts / campaigns / client replies
- Bills / failed charges / renewals / receipts / refunds
- Security alerts / verification codes / suspicious logins / password resets

## Same-Day Review
- Important work messages without time pressure
- Platform/account notices that matter but are not urgent
- Admin items relevant to current priorities

## 16:30 Batch
- Newsletters
- Non-urgent updates
- Routine notifications
- Optional reads

## Ignore / Archive
- Ads
- Promotions
- Repetitive marketing
- Weakly relevant offers
