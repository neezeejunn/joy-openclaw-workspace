# Source Rating Framework

## Purpose
Rate information sources by usefulness so attention is allocated intelligently.

## Dimensions
- Relevance to current priorities
- Signal-to-noise ratio
- Time sensitivity
- Actionability
- Repeat value

## Example Ratings
### Tier A
- High relevance
- High signal
- Often actionable
- Worth surfacing quickly

### Tier B
- Useful but not urgent
- Usually belongs in routine review

### Tier C
- Low-value noise, promotions, repetitive updates
- Batch or ignore unless a rare exception appears

## Rule
A source's rating can change over time. Reassess periodically.
