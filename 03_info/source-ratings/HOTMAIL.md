# Source Rating: Hotmail

## Role
Hotmail is a mixed-value source containing both mission-critical messages and low-value marketing noise.

## Default Rating
- Base rating: Tier B mixed source

## Upgrade to Tier A when
- sender/topic is shipping-related
- sender/topic is work/collaboration-related
- sender/topic is billing-related
- sender/topic is security-related

## Downgrade to Tier C when
- sender/topic is clearly promotional
- content is repetitive marketing
- no actionability and low relevance

## Handling Principle
Treat Hotmail as a source that requires strong triage, not as a source that should interrupt by default.
