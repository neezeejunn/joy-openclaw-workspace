# Hotmail Filtering Rules

## User Priorities
### Highest-value email types
- Shipping / delivery information
- Work / partnership / collaboration
- Bills / payments / subscriptions
- Security alerts / verification / account risk

### Low-value email types
- Ads
- Promotions

## Routing Rules
### Immediate
Send to immediate attention if the message appears to be:
- shipment / tracking / delivery / pickup / delay
- work collaboration / brand / client / proposal / contract / campaign / quote
- billing / payment / invoice / renewal / failed charge / receipt / refund
- security alert / login alert / verification code / password reset / suspicious activity

### Same-Day Review
Use for:
- useful work-related messages that are not urgent
- account or platform notices without immediate risk
- admin items that matter but do not require instant interruption

### 16:30 Batch
Use for:
- newsletters
- routine platform updates
- non-urgent notifications
- optional reading

### Ignore / Archive / Downgrade
Use for:
- ads
- promotions
- repetitive marketing
- low-relevance offers
