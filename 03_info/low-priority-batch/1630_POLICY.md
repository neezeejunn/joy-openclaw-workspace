# 16:30 Batch Policy

## Goal
Protect focus during the day by batching low-priority information into one review window.

## Batch Candidates
- Promotions
- Newsletters with low urgency
- Notifications without immediate action
- Social/media noise
- Optional reading

## Not for Batch
- Time-sensitive decisions
- Messages requiring same-day response
- Financial / administrative items with deadlines
- Anything clearly high-impact

## Output Format
The 16:30 batch should produce:
- short summary of low-priority items
- archive / ignore recommendations
- any optional reads worth saving
