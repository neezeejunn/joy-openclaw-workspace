# YouTube Quality Gates

## Purpose
Decide whether a video's available content is strong enough for a formal five-dimension comparison.

## Gate A: Date Gate
Pass only if:
- the publish date is confirmed
- the video belongs to the same report day / market day

## Gate B: Content Gate
Pass only if at least one of the following is true:
- transcript available
- detailed description available
- rich extraction provides enough detail about arguments, sectors, stocks, and technical levels

## Gate C: Relevance Gate
Pass only if the video is actually a market-analysis video relevant to the current report.

## Gate D: Comparison Depth Gate
A formal five-dimension comparison needs enough content to compare:
- market environment
- sector view
- market data / calendar emphasis
- stock fundamentals
- stock technicals

If this depth is missing, downgrade the output to a shallow comparison note.
