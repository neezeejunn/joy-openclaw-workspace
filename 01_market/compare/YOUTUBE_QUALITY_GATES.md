# YouTube Quality Gates

## Purpose
Decide whether a video's available content is strong enough for a formal five-dimension comparison.

## Gate A: Date Gate
Pass only if:
- the publish date is confirmed
- the video belongs to the same report day / market day

## Gate B: Transcript Gate
Pass only if:
- a same-day transcript is available

If transcript is unavailable, fail immediately.

## Gate C: Relevance Gate
Pass only if the video is actually a market-analysis video relevant to the current report.

## Gate D: Comparison Depth Gate
A formal five-dimension comparison needs enough transcript content to compare:
- market environment
- sector view
- market data / calendar emphasis
- stock fundamentals
- stock technicals

If this depth is missing, do not downgrade to description-based comparison; fail the YouTube comparison step.
