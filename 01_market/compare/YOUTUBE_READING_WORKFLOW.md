# YouTube Reading Workflow

## Goal
Obtain enough same-day video content to support a real comparison report instead of relying on title-only interpretation.

## Preferred Reading Priority
### Level 1: Transcript
Best option.
Use a transcript when available because it provides the richest direct access to the speaker's actual claims.

### Level 2: Detailed description
If no transcript is available, use the full video description if it contains meaningful argument detail.

### Level 3: Rich content extraction
If transcript and description are unavailable, use any sufficiently rich content extraction that captures the video's main claims, structure, and emphasis.

### Level 4: Title only
This is not sufficient for a formal comparison report.
If only the title is available, comparison must be marked provisional / shallow.

## Same-Day Validation
Before comparison, confirm:
- video publish date
- that the video is about the same market day being analyzed
- that the content is a market-analysis video rather than unrelated channel content

## Extraction Questions
When reading a video, extract:
1. What is the market-environment call?
2. Which sectors are favored or avoided?
3. What market events / dates / timing windows are emphasized?
4. Which stocks are mentioned and what is the fundamental logic?
5. Which technical levels / trend arguments are emphasized?
