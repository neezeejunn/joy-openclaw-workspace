# Deliverable Types

## Type 1: Formal Comparison Report
Requirements:
- same-day video found
- same-day transcript available
- same-day market/news sources available
- five-dimension comparison possible

## Type 2: Independent Market Report
Requirements:
- market/news analysis available
- no valid transcript-backed YouTube comparison
- report explicitly marked: no_formal_youtube_comparison

## Type 3: No Report
Use when:
- data quality is too weak
- same-day validation fails broadly
- automation window closes without sufficient inputs

## Rule
Do not force Type 1 when only Type 2 is justified.
