# Market Module Input Specification

To produce a high-quality daily/weekly market report, the system should ingest the following inputs.

## A. Market Data
Minimum:
- S&P 500, Nasdaq, Dow, Russell 2000 daily moves
- Sector ETF moves (e.g. XLK, XLF, XLE, XLV, XLI, XLY, XLP, XLU, XLB, XLRE, XLC)
- Treasury yield / DXY / VIX when relevant

Preferred:
- 1-day / 1-week / 1-month / 3-month relative performance
- Breadth indicators
- Earnings calendar / economic calendar

## B. News Inputs
Minimum:
- Top macro/market headlines of the day
- Major earnings/guidance headlines
- Major policy / rates / regulation developments

## C. External Comparison Inputs
YouTube channel: 视野环球财经
Need one of:
- Video URL
- Transcript
- Title + summary notes
- Screenshot/manual notes from the same-day video

## D. User-Provided Fast Path
If automation is unavailable, user can provide:
- Screenshots of major indexes / watchlist / sector board
- A list of key headlines
- A YouTube link or screenshot from the target channel

## Output Standard
The report should still separate:
1. Facts/data
2. Interpretation
3. Opportunity ranking
4. External comparison
