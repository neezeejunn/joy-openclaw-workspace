# Delivery Preferences

## User Preference
### On successful report generation
- Write the full report to the repo/workspace
- Also send a concise summary to Telegram

### If transcript is unavailable
- Still allow an independent market report when market/news analysis quality is sufficient
- Explicitly mark the result as: no_formal_youtube_comparison

## Rationale
This preserves usefulness even when YouTube transcript retrieval fails, while maintaining strict transcript-only rules for formal comparison.
