# 01_market

This module covers US market research, daily recap, weekly opportunity analysis, and external viewpoint comparison.

## Subfolders
- daily/        Daily market briefs
- weekly/       Weekly opportunity reports
- watchlist/    Sector and stock watchlists
- theses/       Reusable investment theses
- compare/      Comparison notes vs external commentators
- sources/      Input source references and setup notes

## Core Output Order
1. Market regime
2. Sector ranking
3. Stock ideas inside preferred sectors
4. Comparison with external commentary
